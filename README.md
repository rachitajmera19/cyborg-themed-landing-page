# CYBERNEXUS — Cyborg-Themed Landing Page Development

A polished responsive landing page created for the **Cyborg-Themed Landing Page Development** task.

## Project Concept

CYBERNEXUS is a futuristic landing page for a human-machine intelligence interface. Instead of using a random robot image, the website builds a complete cyborg story using:

- A half-human, half-machine hero visual made with CSS
- Neural link and augmented vision HUD cards
- Interactive mission control deck
- Feature modules based on a cyborg system
- Simple architecture flow: Sense → Analyze → Assist
- Responsive layout for desktop, tablet, and mobile

## Tech Stack

- HTML5
- CSS3
- JavaScript
- No paid assets
- No external image dependency

## Key Highlights

- Fully responsive landing page
- Modern dark sci-fi design
- Glassmorphism UI cards
- Animated scan line, radar, cyborg visual, and reveal effects
- Interactive mission mode buttons
- Demo form interaction
- Accessibility-friendly semantic structure
- Reduced motion support
- GitHub Pages ready

## How to Run Locally

Open `index.html` in any browser.

## GitHub Pages Deployment

1. Create a new GitHub repository named `cyborg-themed-landing-page`.
2. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Open repository **Settings**.
4. Go to **Pages**.
5. Select **Deploy from branch**.
6. Choose `main` branch and `/root` folder.
7. Save and wait for the live link.
